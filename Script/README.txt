Progetto Database palestra
Realizzato mediante Postgres versione 2.4.3 e PgAdmin 4 versione 5.1
Realizzato su sistema operativo macOS Big Sur v.11.2.3

Procedimento
1) Creazione del database mediante il file Table.sql
2) Impostare i trigger attraverso il file Trigger.sql prima dell'inserimento dei dati
3) Popolare la tabella mediante il file Dati_Inserimento.sql
4) Interrogare, modificare ed eliminare i dati immessi  mediante il file Query_Interrogazione_Modifica_Eliminazione.txt
